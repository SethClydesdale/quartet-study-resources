# Anki Decks for Quartet Study Resources
This folder contains [Anki](https://apps.ankiweb.net/) decks for the Quartet textbooks. You can find the decks in the [lessons](lessons/) folder.


### How to Download the Decks

You can download the decks online [here](https://sethclydesdale.github.io/quartet-study-resources/help/anki-decks/). The instructions below are for downloading from GitHub.

To download a single deck, you can go to the **lessons** folder, click the deck, and then download it via the download button. 

To download all decks at once, you can [download this repo](https://github.com/SethClydesdale/quartet-study-resources/archive/refs/heads/main.zip), unzip the contents, then navigate to **resources/tools/decks** and open the **lessons** folder to access the decks.


### How to Import the Decks
Once you have your decks, open Anki and click "Import File" (or go to File > Import...) to import your decks. 

That's it, you're done. Now you're ready to study the vocab!
